# Week 2 Ethics, Bias, and Responsible-AI Planning

Week 2 ethics planning focused on ensuring that responsible AI is treated as a core project stream rather than a final-week add-on.

Key points:
- bias must remain visible as a project risk
- the model should not be presented as a substitute for agronomic judgement
- predictions must later be accompanied by responsible-use language
- ethics planning should connect to evaluation, deployment, and reporting

This note keeps LO4 evidence visible during the planning phase.
