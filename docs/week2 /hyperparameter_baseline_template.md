# Week 2 Hyperparameter Baseline Template

Week 2 optimisation planning established placeholders for later baseline training.

Planned structure:
- optimizer: to be fixed before first official baseline run
- scheduler: to be fixed before first official baseline run
- epoch budget: practical window rather than unlimited training
- early stopping: required to avoid uncontrolled runs
- checkpoint policy: save best-performing model state

Purpose:
To ensure Week 3 baseline training begins from a controlled plan rather than ad hoc trial-and-error.
